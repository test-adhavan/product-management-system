# Product Management System

A Spring Boot based backend application for managing products and categories.

## Features
- CRUD operations for Products and Categories
- Pagination for product listing
- DTO pattern with layered architecture
- Validation and global exception handling
- RESTful APIs
- Swagger API documentation

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- Maven

## API Endpoints (Sample)
- POST /product
- GET /product?page=0&size=3
- GET /product/{id}
- PUT /product/{id}
- DELETE /product/{id}
- GET /product/category/{id}

- IntelliJ IDEA
